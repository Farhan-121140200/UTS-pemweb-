Ini adalah halaman utama yang berisi menu navigasi untuk berpindah ke halaman lain.Hypertext Markup Language (HTML) adalah bahasa yang digunakan untuk membuat dan mendefinisikan struktur dasar halaman web. HTML terdiri dari elemen-elemen yang disusun dalam bentuk tag, seperti html>, head>, body> dan lain-lain. Elemen-elemen ini digunakan untuk mengatur tampilan dan konten halaman web dan memberikan instruksi kepada browser tentang cara menampilkan informasi.

Setiap halaman web HTML dibuat menggunakan hierarki elemen. Struktur dasar dokumen HTML mencakup elemen berisi (berisi informasi meta seperti judul halaman dan link ke CSS) dan body> (berisi konten yang akan ditampilkan pada halaman). Elemen HTML dapat diberi atribut yang memberikan informasi tambahan tentang elemen tersebut, seperti menunjukkan gambar mana yang akan ditampilkan.

HTML memegang peranan penting dalam membangun pondasi sebuah website, namun untuk menata tampilan dan gaya halaman digunakan CSS (Cascading Style Sheets) secara bersamaan. HTML juga mendukung berbagai fitur interaktif melalui elemen seperti formulir dan hyperlink. Dengan memahami HTML, seseorang dapat membuat website yang responsif dan mudah diakses oleh berbagai perangkat dan browser.
